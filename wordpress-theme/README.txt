== Description ==

Pasiones Platform Theme es un theme WordPress profesional diseñado específicamente para redes sociales de profesionales. Incluye soporte completo para Elementor, integración con el plugin Pasiones Platform y todas las funcionalidades necesarias para crear una plataforma de profesionales con videochat, streaming y monetización.

**Características Principales:**

* 100% compatible con Elementor
* Responsive y mobile-first
* Integración completa con plugin Pasiones Platform
* Templates para Custom Post Types (Profesionales, etc.)
* Sistema de membresías visualizado
* Widgets de sidebar personalizados
* Customizer avanzado
* SEO optimizado
* Diseño moderno y profesional
* Soporte para WooCommerce (opcional)
* Traducible con WPML

**Funcionalidades Específicas:**

* Templates para perfiles de profesionales
* Archive templates con filtros avanzados
* Sistema de valoraciones y reviews
* Badges de membresía (Oro, Plata, Bronce, Gratis)
* Indicadores de estado online
* Tarjetas de profesionales
* Sistema de categorías
* Multi-país y provincias
* Social sharing integrado
* Breadcrumbs personalizados

== Installation ==

1. Sube la carpeta `pasiones-theme` a `/wp-content/themes/`
2. Activa el theme desde el panel de WordPress
3. Instala el plugin "Pasiones Platform" (requerido para funcionalidad completa)
4. Personaliza desde Apariencia → Personalizar
5. Configura los menús desde Apariencia → Menús
6. Agrega widgets a las áreas de widgets

== Frequently Asked Questions ==

= ¿Necesito el plugin Pasiones Platform? =

Sí, para funcionalidad completa. El theme está diseñado para trabajar en conjunto con el plugin. Sin el plugin, algunas features no estarán disponibles.

= ¿Es compatible con Elementor? =

Sí, 100% compatible. Puedes usar Elementor para diseñar páginas completas y el theme incluye soporte para header y footer de Elementor.

= ¿Qué navegadores soporta? =

Todos los navegadores modernos: Chrome, Firefox, Safari, Edge. Internet Explorer no está soportado.

= ¿Es responsive? =

Sí, completamente responsive y optimizado para mobile, tablet y desktop.

= ¿Puedo usar otros plugins de membership? =

El theme está optimizado para el sistema de membresías del plugin Pasiones Platform, pero es compatible con otros plugins.
